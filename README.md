# Lumi - AI Personal Assistant

Lumi is an AI-powered personal assistant designed to help with daily tasks through voice commands and natural language processing.

## Features

- **Voice Interaction**: Speech-to-text and text-to-speech capabilities
- **Wake Word Detection**: "Lumi", Responds to custom wake words
- **Intent Recognition**: Understands and processes user intents
- **Modular Skills**: Extensible skill system for various functionalities
- **State Management**: Maintains conversation context

## Skills Included

- Weather information
- Time and date queries
- System control (shutdown, restart, etc.)
- Application launching
- Screenshot capture

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Run the assistant: `python app/main.py`

## Project Structure

- `app/`: Main application code
- `skills/`: Modular skill implementations
- `core/`: Core processing components
- `audio/`: Audio processing modules
- `utils/`: Utility functions
